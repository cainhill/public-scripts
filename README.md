# Public Scripts

## For calendar management
Script | Purpose
--- | ---
pullTasksForward.gs | Pulls overdue calendar tasks to the next available calendar slot.
pushTasksToNextBucket.gs | Pushes overdue tasks to the next task bucket.
colorCodeEvents.gs | Highlights events as yellow for tasks and red for upcoming events that are missing key information.
